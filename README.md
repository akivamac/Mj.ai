# Monkey Joe 🐒

A rules-based AI assistant. No LLM — his brain lives in `brain/`.

## Structure
- `index.html` — main chat UI
- `settings.html` — brain editor + MCP config
- `style.css` — styles
- `js/` — modules
- `brain/` — JSON knowledge base

## Run
Open `index.html` in a browser. No server needed.

## Brain
Edit Joe's knowledge in Settings or directly in the JSON files.
